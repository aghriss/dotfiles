## Config files

#### Neovim: NvChad (for NvChad v2.0)

`./NvChad` path should be mapped to `~/.config/nvim/lua/custom` after you install 
[NvChad](nvchad.com)
```bash
ln -s ./NvChad ~/.config/nvim/lua/custom/
```

### Z shell Config

- `./zshell/zsh` -> `~/.config/zsh`
- `./zshell/zshrc` -> `~/.zshrc`

### tmux

Install Tmux TPM to ~/.config/tmux/plugins/tpm
